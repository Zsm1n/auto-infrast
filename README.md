# auto-infrast

🚀 基于 Python 的《明日方舟》基建自动化规划程序。

厌倦了手动安排基建干员？这个工具能根据你的干员库存和具体练度，为你智能生成最优的基建排班方案。

## 核心功能

📊 **干员数据导入**: 支持通过MAA干员识别导出的 JSON 数据自动识别你的干员图鉴与技能等级。

🧠 **智能规划算法**: 采用优化算法，自动分配制造站、发电站、会客室等设施的干员，以生产力为核心目标。

⚙️ **个性化配置**: 可指定偏好的贸易站类型（如赤金/龙门币）、无人机使用策略等。

📈 **收益报告**: 生成清晰的规划报告，展示预期收益与排班详情。

## 它能做什么？

✔️ 分析你的干员box和技能

✔️ 自动排班制造站、发电站、会客室等

✔️ 生成一目了然的排班表

✔️ 让你从繁琐的基建管理中解脱出来！
